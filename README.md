#  Personal Art Portfolio Website

A modern, responsive and visually engaging **artist portfolio website** built using **HTML, CSS and JavaScript**.  
This website showcases artworks created by **Ashvini**, including charcoal sketches, graphite portraits, and colorful illustrations.  
It includes a gallery, category filters, lightbox preview, and an about & contact section.
